Comment Accéder aux Ressources ?

Le bot LeSéminaire vous permet d'accéder et de contribuer à la base de données de ressources directement depuis Discord.

Consulter les Ressources
!resource list [catégorie]

Affiche les ressources disponibles, filtré par catégorie optionnelle.

!resource search [terme]

Recherche des ressources par mot-clé.

!resource categories

Affiche la liste des catégories disponibles.

Contribuer aux Ressources
!resource add [titre] [URL] [catégorie] [description]

Ajoute une nouvelle ressource à la base de données.

Partagez vos trouvailles avec la communauté pour enrichir notre base de connaissances commune !

Vous avez une ressource à partager ?

Utilisez le bot sur Discord ou contactez-nous pour l'ajouter à notre base de données !
